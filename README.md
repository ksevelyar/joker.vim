# Joker

Dark colorscheme for Vim/Neovim inspired by [vim-gotham](https://github.com/whatyouhide/vim-gotham).

## Install

### [vim-plug](https://github.com/junegunn/vim-plug)
```
call plug#begin()
Plug 'ksevelyar/joker.vim'
call plug#end()

colorscheme joker
```
